Destaque · Machine Learning

Previsão de Preços de Carros com ML

Modelo preditivo de regressão treinado para estimar o preço de venda de automóveis com base em características técnicas e comerciais. Projeto end-to-end cobrindo análise exploratória, feature engineering, treinamento e avaliação de métricas.

  • Pipeline completo: EDA → pré-processamento → modelagem → avaliação
  • Encoding de variáveis categóricas e normalização de features numéricas
  • Identificação dos fatores com maior impacto no preço (motorização, combustível, dimensões)
  • Avaliação com métricas R², MAE e RMSE para validar a capacidade preditiva
  • Visualizações de correlação e distribuição para suporte às decisões de feature selection
